# mallvipmanage 樱诺商城
一个基于Java Model2 开发模式开发的商城管理系统。
后端使用了Java Servlet技术处理HTTP请求，加上JSP技术，使用EL表达式和JSTL标签来辅助开发
前端页面使用了Jquery、BootStrap框架
